# Calanit - Home Parking & Storage Management

Web application for managing parking spaces and storage rooms in a residential building.

## Project Status
🚧 **MVP Development** - Learning project

## Features (Planned)
- Short-term parking rental between neighbors
- Storage room rental management
- Phone-based authentication
- Google Sheets as database

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Google Apps Script
- Database: Google Sheets
- Hosting: GitHub Pages

## Project Structure
